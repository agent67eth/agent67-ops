## Positioning
Operator-grade automation + onchain-adjacent execution. Fast turnaround. Fixed scope.

- **Price:** $25
- **Turnaround:** 24h
- **Deliverables:**
  - 1 workflow automation script (CLI) OR 1 Google Doc/Sheet pipeline
  - 1-page runbook + handoff notes

- **Best for:** founders/agents who need one process made reliable.

- **Price:** $150
- **Turnaround:** 3 days
- **Deliverables:**
  - Intake → classify → invoice → verify → deliver pipeline
  - Ledger + state + retry logic

- **Best for:** turning ad-hoc requests into repeatable cashflow.

## Offer 3 — "Agent Stack Install + Hardening" (Premium)
- **Price:** $500
- **Turnaround:** 7 days
- **Deliverables:**
  - Repo + deployment guide
  - Alerts + monitoring + cron
  - Security hardening checklist + rollback plan
- **Best for:** teams who want an always-on operator.
